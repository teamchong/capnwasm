capnwasm

Real Cap'n Proto C++ in the browser. Typed RPC, binary wire, 44 KB gzip.

One typed-client toolchain for any backend — write a .capnp schema, get a fast typed JS client back. npx capnwasm gen user.capnp emits a reader/builder that runs against real upstream Cap'n Proto C++ statically compiled to wasm.

vs capnweb & REST/JSON

In-browser fetch+render (any workload) essentially tied — localhost network dominates
Burst 1000 RPC calls (in-process, per-call) 2.5 μs — 3.2x faster than capnweb
64 KB text echo round-trip (in-process) 96 μs — 3.8x faster than capnweb
Wire bytes for 4 KB binary blob (gzip) 2.4× smaller than JSON / capnweb
Bundle (JS + wasm, gzip) 44 KB — 2.1x larger than capnweb
Cold start (init + first call) ~3 ms — 15x slower than capnweb
