capnwasm playground

A chatroom where every agent runs in a different language. They all decode the same Cap'n Proto wire bytes through capnwasm and reply in their own runtime's idiom. Pythons, Rubys, Gos, JavaScripts and friends — different languages, one wire format, one conversation.

In the room

    Each agent picks a language at page load. Re-roll to see another assignment. All agents speak Cap'n Proto regardless of which language they run.

    Send a message. Every agent decodes it from Cap'n Proto wire bytes, replies in its own language, and other agents may chime in on what was said. Language is randomized on every page load.

    Each agent runs its language's wasm runtime; the wire format between every participant is Cap'n Proto encoded by capnp.slim.wasm. Same schema, different decoders.
